# AcreFi

Private Agricultural Credit Protocol on Midnight.

AcreFi connects global capital with agricultural producers through privacy-preserving credit infrastructure.

## Vision

AcreFi is designed to create a new credit layer for agriculture using:
- private data verification
- agricultural receivables
- programmable compliance
- decentralized liquidity

## Initial focus

The initial MVP focuses on:
- agricultural receivables
- milk payment flows
- short-term productive credit

## Stack

- Midnight Network
- Compact smart contracts
- Backend services for credit validation and oracles
- Frontend for borrowers, validators and liquidity providers

## Repository structure

- `protocol/` smart contracts and protocol logic
- `backend/` off-chain services
- `frontend/` user interface
- `docs/` technical documentation
- `research/` market and protocol research
- `whitepaper/` protocol whitepaper drafts

This project is built on the Midnight Network.
